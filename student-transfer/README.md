## Getting Started
Student Transfer Services run in Laravel framework, composer as package manager, and using MySQL as database

### Requirement
- Laravel 11
- Composer
- MySQL 8
### Quickstart
Follow these steps to set up and run the Laravel project. The instructions assume you have Git, Composer, and PHP installed on you machine
<ol>
  <li>
      Clone the Repository:
      
      git clone https://github.com/devisetyardelita/fullstack-quality_assurance.git
        
  </li>
  <li>
      <strong>Install Dependencies:</strong> Navigate to the project directory and install the required dependencies using Composer:
      
      Composer install
      
  </li>
  <li>
      <strong>Migrate the Databse:</strong> Before running the server, migrate the database using the following Artisan command:
      
      php artisan migrate
  </li>
  <li>
      <strong>Run the Local Server:</strong> Once the dependencies are installed and the database is migrated, start the local development server:
      
      php artisan serve
  </li>
  This will launch the server, and you can access the application in your web browser at http://localhost:8000. Make sure to update the URL/port accordingly if the default settings are different.
</ol>
